Opdracht 4c

Tekst

C. Extend the code so that one LED slowly fades in and the other fades out at the same time.

Code


int greenLedPin = 10;
int yellowLedPin = 9;

void setup(){
	pinMode(greenLedPin, OUTPUT);
	pinMode(yellowLedPin, OUTPUT);
}

void loop(){
	for (int brightness=0; brightness < 256; brightness++){
		analogWrite(greenLedPin, brightness);
		analogWrite(yellowLedPin, 256 - brightness);
		delay(10);
	}

	for (int brightness=256; brightness >= 0; brightness--){
		analogWrite(greenLedPin, brightness);
		analogWrite(yellowLedPin, 256 - brightness);
		delay(10);
	}
}

Opdracht 7

Tekst

1. Use the code you wrote in exercise 6A. Comment out the line you wrote to scale the sensor value. Do this by adding // at the beginning of the line. This line is now temporarily disabled. 2. Verify the code and upload to the board. 3. Open the Serial Monitor. What you see now is the amount of light that the LDR perceives expressed in a number. Under ideal circumstances this range is between 0 and 1023. But in practice, the measured range is a lot smaller. If you hold you hand over the LDR, you should see a drop in this value. Write down the highest (unobscured) and lowest (obscured) value. This the actual range the sensor has under these lighting conditions. 4. Uncomment the line you commented and change it to rescale the sensor value in the proper way. The object here is to rescale the actual range you just wrote down to the 0-255 range the LED’s accept. (You could also use the map() function if this makes things easier) 5. Verify the code and upload to the board. If all went well, the LEDs should now fade according to the amount of light that the LDR perceives.

Code


float sensorValue = 0;
float sensorPin = A0;

int greenLedPin = 10;
int yellowLedPin = 9;

void setup (){
	pinMode(sensorPin, INPUT);
	Serial.begin(9600);

	pinMode(greenLedPin, OUTPUT);
	pinMode(yellowLedPin, OUTPUT);
}

void loop(){
	sensorValue = analogRead(sensorPin);

	sensorValue = (sensorValue/255)*255;

	analogWrite(greenLedPin, sensorValue);
	analogWrite(yellowLedPin, sensorValue);

	Serial.println(sensorValue);

	delay(20);
}

import processing.serial.*;

Serial myPort;
String sensorReading="";

void setup() {
	size(800, 800);
	myPort = new Serial(this, Serial.list()[1], 9600);
	myPort.bufferUntil('\n');
}

void draw() {
	background(255);
	fill(0);
	text("Sensor Reading: " + sensorReading, 20, 20);
	triangle(2, 2, 2, 2, float(sensorReading), float(sensorReading));
}

void serialEvent (Serial myPort) {
	sensorReading = myPort.readStringUntil('\n');
}

Opdracht 10f

Tekst

F. The LED lights up when one button of the two buttons is pressed, but not both.

Code


int ledPin = 13;  
int buttonPinL = 3;
int buttonPinR = 2; 

int buttonState = 0;  

void setup() {
	pinMode(ledPin, OUTPUT);
	pinMode(buttonPinL, INPUT);
	pinMode(buttonPinR, INPUT);
}

void loop() {
	if (digitalRead(buttonPinL) && digitalRead(buttonPinR)) {
		digitalWrite(ledPin, LOW);
	} else {
		digitalWrite(ledPin, digitalRead(buttonPinR) || digitalRead(buttonPinR));
	}
}

	delay(10);
